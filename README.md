## ✨ 博客里有什么

这个博客的起点其实挺随意的——最早是拿别人的源码搭着玩玩，直到2025年9月才开始正经建设和写内容，后来又重构了一版，才变成你现在看到的样子。所以严格来说，它还是个年轻的博客，内容也在慢慢积累中。

目前主要分这么几块：

- **技术笔记**：工作中踩过的坑、项目复盘、AI相关技术栈的学习记录（大模型应用、前后端工程化、部署落地等方向）
- **工程实践**：从"能跑就行"到"可维护、可扩展"的意识转变，包括架构思考、代码规范、协作流程这些课堂上不太教的东西
- **职场观察**：刚工作这段时间的真实感受——怎么理解需求、怎么和团队沟通、怎么在deadline面前保持冷静（虽然还没完全做到）
- **生活随笔**：工作之余的一些阅读、想法和碎碎念，毕竟人不能只活在代码里

内容不算多，但每一篇都是真实的记录。不美化，也不贩卖焦虑。

---

## 🎯 为何于此

最开始搭博客纯粹是觉得好玩，后来发现把自己写过的代码、踩过的坑整理成文字，比在脑子里过一遍要清晰得多。慢慢就养成了习惯——遇到值得记的东西就写下来，不为流量，就当是给自己留一份底稿。

从玩票到认真对待，这个博客见证了我从考研失利、到找实习、再到入职工作的全过程。以后回头看，应该会觉得挺有意思吧。
